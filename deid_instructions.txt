Testing DeIdentification.jl

Quick Notes:

- Make sure you are on Julia 1.0 (maybe 0.7 will work as well)

- In your directory make sure to have 3 sub-directories: data, logs, output
	- Idea for Paul: maybe create those folders if they do not exist?

- When creating the DeIdentified CSVs, make sure to use the following modules to avoid errors: CSV, DataFrames, Dates, YAML, Random, Memento, JSON, SHA

- The DeIdentified CSVs will be in the output directory


To DeIdentify (see src/make_deid.jl):

using DeIdentification

using CSV
using DataFrames
using Dates
using YAML
using Random
using Memento
using JSON
using SHA

data_dir = --PATH TO DATA--
working_dir = --PATH OF SOURCE--

cd(data_dir)
cfg = YAML.load(open("../test/ehr_data.yml"))
println(cfg)

proj_config = DeIdConfig("../test/ehr_data.yml")
deid = DeIdentified(proj_config)
write(deid)

cd(working_dir)
